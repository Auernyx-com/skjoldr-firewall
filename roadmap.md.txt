# Skjoldr Firewall – Roadmap

## Short Term (1.x Series)

- **1.1.0** – Version Awareness  
  → Version banner in GUI + CLI  
  → Structured changelog & roadmap

- **1.2.x** – Visibility & Logging  
  → Built-in log viewer in GUI  
  → Export current firewall rules to file  
  → Safer reset confirmation dialogs

- **1.3.x** – Smart Presets  
  → Profile-specific modes (Home / Public Wi-Fi / Travel)  
  → App-based allow lists (browser, Steam, etc.)

## Medium Term
- Single-click EXE wrapper (no PowerShell dependency)
- MSI installer + silent deployment support
- JSON/YAML config import/export
- Toast notifications on mode change

## Long Term (2.x+)
- Centralized logging & rule diff viewer
- Multi-machine profile sync
- Integration with Windows Event Log
- Hardening templates per threat model
